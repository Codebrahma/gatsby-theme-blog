## Clone the repository
```
  git clone git@github.com:sachinmjadhav/gatsby-blog-theme.git
```

## Install all the dependencies
```
  cd gatsby-theme-blog
  yarn
```

## Run the dev server
```
  yarn workspace site develop
```
